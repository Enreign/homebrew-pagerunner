# Homebrew Pagerunner Tap

Custom Homebrew tap for [Pagerunner](https://github.com/Enreign/pagerunner) — Chrome browser automation MCP server for AI agents.

## Installation

```bash
brew tap enreign/pagerunner
brew install pagerunner
```

## Usage

```bash
# Initialize configuration
pagerunner init

# Run as MCP server
pagerunner mcp

# Check version
pagerunner --version

# List available commands
pagerunner --help
```

## Requirements

- macOS 10.13+ or Linux
- Chrome or Chromium browser installed
- Requires closing Chrome browser windows using the profile before connecting

## Updating

To upgrade to the latest version:

```bash
brew upgrade pagerunner
```
